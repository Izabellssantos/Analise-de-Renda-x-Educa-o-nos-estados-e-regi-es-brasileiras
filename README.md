## 📌 Contexto

## 🧠 Insights

A análise foi realizada a partir de tabelas dinâmicas e visualizações em dashboard, permitindo identificar padrões nos indicadores de renda e educação entre os estadose regiões brasileiras.
Os dados indicam uma tendência de relação positiva entre renda e educação, na qual estados com melhores níveis educacionais tendem a apresentar maiores índices de renda. Ainda assim, essa relação não é uniforme, sugerindo a influência de outros fatores socioeconômicos.
O Distrito Federal se destaca por apresentar os maiores índices, configurando-se como um outlier (ponto fora da curva), uma vez que seu desempenho não representa necessariamente o padrão da região Centro-Oeste como um todo.
Na análise regional, observa-se que a região Sul apresenta os maiores índices médios tanto de renda quanto de educação, destacando-se como a região de melhor desempenho geral. Em seguida, as regiões Sudeste e Centro-Oeste apresentam valores semelhantes no indicador de renda (0,68), diferenciando-se no nível educacional, no qual o Sudeste apresenta desempenho superior.
As regiões Norte e Nordeste concentram os menores índices, evidenciando disparidades regionais significativas no país.
Destaca-se ainda que, no Centro-Oeste, há influência relevante do Distrito Federal, cujo desempenho elevado impacta a média regional, caracterizando um efeito de concentração.
Por fim, ao observar os dados agrupados por região, percebe-se relativa homogeneidade interna, com estados apresentando valores próximos entre si, o que reforça a existência de padrões regionais consistentes.
O dashboard permite identificar rapidamente essas disparidades e pode servir como base para análises mais aprofundadas sobre políticas públicas e desenvolvimento regional.

## 📌 Conclusão
Os dados evidenciam desigualdades regionais no Brasil, com destaque para Sul e Sudeste.
